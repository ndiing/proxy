## Server.listen()

### Syntax
```
listen(port,hostname,backlog)
```

### Parameters
<dl>
    <dt><code>port</code></dt>
    <dd>A <code>Number</code> , The default is <code>8888</code></dd>
    <dt><code>hostname</code></dt>
    <dd>A <code>String/Function</code> , The default is <code>127.0.0.1</code></dd>
    <dt><code>backlog</code></dt>
    <dd>A <code>function</code> </dd>
</dl>

### Return value

<dl>
    <dt>None <code>undefined</code></dt>
</dl>
